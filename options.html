Azure DevOps target

The org segment of the URL: dev.azure.com/<org>/…
Plain text — no URL-encoding. Spaces are fine (e.g. "Design Organization").
The path below the project, used to scope searches. Don't include the project name. Mind the casing (e.g. lowercase "designOrg").

Changing the target affects new links only. Items already saved in your popup History (and existing quick links and bookmarks) keep the full URL they were created with.
